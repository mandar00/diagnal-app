## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mandar00/diagnal-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your BASE_URL in `.env.developmnt` and `.env.production`
   ```js
   const BASE_URL = "ENTER YOUR API";
   ```

### Start

- Run project locally
  ```sh
  npm run dev
  ```
